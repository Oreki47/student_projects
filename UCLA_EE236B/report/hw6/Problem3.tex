\subsection*{T4.43}
\subsubsection*{(b)}
From the property that $\lambda_1 \leq t$ if and only if $A(x)\preceq tI$. The problem can be formed as 
\begin{align*}
&minimize \qquad t-d\\
&subject\ to \qquad dI \preceq A(x) \preceq tI
\end{align*}

\end{align*}
